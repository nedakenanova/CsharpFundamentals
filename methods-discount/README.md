# Основи на C# – Методи

Това хранилище съдържа задача от  **Методи в C#**.

## Задача: Изчисляване на крайна цена с отстъпка

Потребителят въвежда:
- Начална цена
- Процент отстъпка

Програмата:
- Изчислява крайната цена след отстъпка
- Показва колко е спестено

### Пример:
Въведи цена: 100
Въведи отстъпка %: 20
Крайна цена: 80.00
Спестяваш: 20.00
